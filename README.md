# John Bass: Extreme Bass Plugin for Audacity

This is a port of the **John Bass** JSFX from REAPER, adapted specifically for Audacity. It includes additional features such as Sub-Bass Injection and Stereo Widening to suit the destructive editing workflow.

### Features
*   **Destruction Amount:** Heavy distortion and clipping.
*   **Sub-Bass Injection:** Adds a distorted 60Hz rumble to the signal.
*   **Stereo Widening:** Expands the stereo field (requires a stereo track).
*   **Presets:** 
    *   Mild Fry, Deep Fried Meme, Broken Android, Nuclear Option.
    *   **Swim44:** Instantly silences the audio.
*   **Low Quality Mode:** Bitcrushing effect.
*   **The Ear Piercer:** Optional high-frequency beep.

### Installation
1.  Download the `.ny` file.
2.  Locate your Audacity **Plug-Ins** folder:
    *   **Windows:** `C:\Program Files\Audacity\Plug-Ins`
    *   **macOS:** `~/Library/Application Support/audacity/Plug-Ins`
    *   **Linux:** `~/.audacity-data/Plug-Ins`
3.  Place the file into the folder.
4.  Open Audacity and go to **Tools** ‚Üí **Plugin Manager**.
5.  Find **John Bass**, select it, and click **Enable**.

### How to Use
1.  Select your audio track.
2.  Go to the **Effect** menu and select **John Bass**.
3.  **Note:** To use the sliders (Destruction, Sub-Bass, Width), you must select **"Manual Mode"** in the preset dropdown. Other presets will override slider positions.
